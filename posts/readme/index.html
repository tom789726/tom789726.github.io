A few thoughts

  • Static site generator
    • Hugo / Jekyll
  • Hugo: command line 自動生網頁，有幾種安裝方法
    • Pre-built binaries
    • Through package manager $\leftarrow$ current method

Step1: Install Chocolatey

  • Windows原來都可以裝Package Manager $\rightarrow$ 隨便選了Chocolatey
  • My first package manger in Windows :)
  • Install: 用Windows內建PowerShell (admin)
    1. Run Get-ExecutionPolicy $\rightarrow$ if it returns restricted, run Set-ExecutionPolicy AllSigned
    2. Run Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
    3. Run choco -? / choco 檢查安裝成功or not
    4. Reference: link

  • Version: v0.11.3

Step2: Install Hugo

  • 既然有Chocolatey就用啦! :D
  • choco install hugo -confirm :x:
    • Update: 若果只裝hugo，後面build site會有error
  • choco install hugo-extended -confirm

Hugo

  • Version: v0.89.1

Hugo (extended)

  • Version: v0.89.1

Step3: Create New Sites & Add Theme (Quick Start)

  • 以下用返Git bash
  • reference: link

Build new site

  1. cd 返去Desktop
  2. hugo new site qstart

Apply template

  • Example: ananke
  1. cd quickstart
  2. git init
  3. git git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
  4. echo theme = \"ananke\" >> config.toml

Add Content & Publish (server)

  1. hugo new posts/my-first-post.md
  2. 隨便改一改內容 & 準備好發佈時要改post入面的header
    • draft: false
  3. hugo server -D
    • :x: 會連不上，改用: hugo server --theme=anankin
  4. 打開http://localhost:1313/就會睇到網站

Customize the theme Site Configuration

  • 打開config.toml
baseURL = 'http://example.org/'
languageCode = 'en-us'
title = 'My New Hugo Site'
theme = "ananke"
  • 自己改想改的東西: e.g. title
  • baseURL
    • 如果你有自己的網站domain，可以改
    • 如果用local server就會skip

Build static site

  • hugo -D
    • 檔案會放在public/
    • 可以加-d / --destination改發佈位置
    • or set publishdir in config file (config.toml)

Step4: Publish on GitHub

  • 去Github開一個repo，一定要用<user-name>.github.io
  • 回到本地資料夾，將剛剛產生的static site push上去
cd public
git init
git add .
git commit -m "First Commit"
git remote add origin https://github.com/tom789726/tom789726.github.io.git
git push -u origin master
  • DONE!，應該等一等就可以上tom789726.github.io睇做好的網頁了
  • 另外亦可以將完整project掉上去Github，隨心!
